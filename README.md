# TJJupiter Demo (iOS)

This project is a sample iOS application demonstrating how to use **TJJupiterSDK** for indoor positioning and navigation.

It provides a simple UI to start/stop the Jupiter service and visualize real-time positioning results.

---

## ✨ Features

- 🚀 Start / Stop Jupiter service
- 📡 Real-time positioning result display
- 🏢 Building / Level / Coordinates visualization
- 🧭 Navigation destination & routing test
- 🧪 Simulation mode support

---

## 📦 Requirements

- iOS 15.0+
- Xcode 14+
- Swift 5.0+

---

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/tjlabs/TJJupiter-demo-ios.git
cd TJJupiter-demo-ios
```

### 2. Clone the repository
```bash
pod install
```

### 3. Open workspace
```bash
open TJJupiterDemo.xcworkspace
```

### 4.Configure API Keys
```bash
TJJupiterAuth.shared.auth(
    accessKey: "YOUR_ACCESS_KEY",
    secretAccessKey: "YOUR_SECRET_ACCESS_KEY"
)
```

```bash
pod install
```
